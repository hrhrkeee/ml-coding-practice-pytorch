# 11. 高度な最適化アルゴリズム (Advanced Optimizers)

適応的学習率を持つ高度な最適化アルゴリズムを実装します。

## 含まれるノートブック

1. **01_adam.ipynb** - Adam (Adaptive Moment Estimation)
   - 1次モーメントと2次モーメント
   - バイアス補正
   - 最も広く使われる最適化手法

2. **02_rmsprop.ipynb** - RMSProp (Root Mean Square Propagation)
   - 勾配の2乗の移動平均
   - パラメータごとの適応的学習率
   - RNNの学習に効果的

## 学習内容

- 適応的学習率の利点
- モーメンタムベースの最適化
- 各最適化手法の特徴と使い分け
- ハイパーパラメータの設定
