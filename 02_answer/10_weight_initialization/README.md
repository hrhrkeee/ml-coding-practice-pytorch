# 10. 重み初期化 (Weight Initialization)

ニューラルネットワークの重みの初期値設定手法を実装します。

## 含まれるノートブック

1. **01_xavier.ipynb** - Xavier初期化 (Glorot初期化)
   - Xavier Uniform初期化
   - Xavier Normal初期化
   - Sigmoid/Tanh活性化関数に適した初期化

2. **02_he.ipynb** - He初期化
   - He Uniform初期化
   - He Normal初期化
   - ReLU系活性化関数に適した初期化

## 学習内容

- 重み初期化の重要性
- 勾配消失爆発問題
- 活性化関数に応じた初期化手法
- 分散の維持による安定した学習
